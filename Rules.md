# 바이브 코딩용 PRD 템플릿

## 1. 프로젝트 개요
- 프로젝트 이름: 멤버 리스트 대시보드 (Member List Dashboard)
- 한 줄 소개: 부서별 멤버 카테고리화, 정렬, 무한 스크롤 및 추가/수정 기능이 탑재된 세련된 다크/라이트 모드 지원 멤버 관리 UI
- 목표 사용자: 팀/조직 관리자 및 조직원
- 해결하려는 문제: 직관적이고 세련된 화면에서 멤버 정보를 확인하고, 부서별 필터링과 정렬을 통해 원하는 직원을 신속하게 검색하며, 임시 멤버 추가/수정이 가능한 컴팩트 대시보드 구축

## 2. 핵심 기능
- 기능 1: 다크/라이트 모드 토글 테마 지원 및 모던 그라데이션 UI
- 기능 2: 부서별 카테고리 탭 분류 및 이름/부서순 정렬 기능
- 기능 3: 무한 스크롤 시뮬레이션 (스크롤 최하단 시 멤버 추가 로드)
- 기능 4: 간단한 멤버 추가 및 수정 폼 인터페이스 (React State 기반)

## 3. 화면/흐름
- 진입 화면: 대시보드 상단에 다크/라이트 모드 스위치, 검색바, 정렬 옵션, 그리고 부서 필터 탭이 위치하고, 아래에 멤버 카드 그리드가 노출됨. 우측 상단에 '멤버 추가' 버튼 배치.
- 주요 화면:
  - 멤버 리스트 그리드: 각 멤버의 프로필 이미지(스타일리시한 그래디언트 아바타), 이름, 부서, 직무 태그, 이메일, 연락처, 소셜 아이콘이 나타남.
  - 멤버 추가/수정 모달 폼: 이름, 직무, 부서, 이메일, 전화번호, 소셜(GitHub) 링크 등을 입력 및 수정할 수 있는 모션 모달.
- 사용자 행동 흐름:
  - 부서 탭 클릭 -> 해당 부서 멤버 필터링
  - 검색창 입력 -> 실시간 이름 및 직무 검색 필터링
  - 하단 스크롤 -> 무한 스크롤 형태로 추가 멤버 데이터를 자동 로딩
  - '멤버 추가' 버튼 클릭 -> 모달 생성 -> 정보 입력 -> 리스트 반영
  - 카드 내 '수정' 아이콘 클릭 -> 모달에 기존 정보 로딩 -> 수정 완료 -> 리스트 반영

## 4. 디자인/UX 가이드
- 톤앤매너: Sleek & Modern, Premium, Clean
- 색상: HSL 기반의 세련된 다크/라이트 테마 색상 (블루/퍼플/인디고 포인트 그라데이션)
- 폰트: Inter / Outfit 등 깔끔한 산세리프 서체 적용 (Google Fonts 연동)
- 버튼/입력폼 스타일: 미세한 섀도우, 둥근 모서리 (border-radius: 12px~16px), 포커스 시 네온 테두리 광원 효과, 부드러운 호버 트랜지션

## 5. 기술 제약사항
- 사용 기술: React 19, Vite, Vanilla CSS
- 브라우저/환경: 모던 웹 브라우저 (Chrome, Safari, Firefox 등)
- 데이터 저장 방식: 임시 React State 관리 (데이터 저장 방식은 추후 논의)
- API 필요 여부: 없음 (로컬 Mock Data 및 난수화 기반 생성 데이터 활용)

## 6. 완료 기준
- 기능이 정상 동작해야 함 (필터, 정렬, 무한 스크롤, 추가, 수정)
- UI가 자연스럽게 연결되어야 함 (다크/라이트 전환, 모달 팝업 및 슬라이드 애니메이션)
- 에러 없이 실행되어야 함

## 7. AI에게 전달할 지시문
- 구현할 기능을 단계별로 설명해줘
- 코드 구조를 간단히 설명해줘
- 핵심 컴포넌트와 파일 이름을 제안해줘
- 테스트나 실행 방법도 함께 알려줘
- 식별자는 카멜케이스를 사용한다.
