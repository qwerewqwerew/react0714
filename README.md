#### 깃으로 관리하기

- `git init`
  - !현재 폴더를 Git 저장소로 초기화합니다.
- `git add .`
  - README.md 파일을 다음 커밋에 포함될 파일로 선택합니다.
- `git commit -m "first commit"`
  - "커밋 메시지"이라는 메시지로 변경 내용을 저장합니다.
- `git branch -M main`
  - !현재 브랜치 이름을 main으로 변경합니다.
- `git remote add origin https://github.com/qwerewqwerew/react-test.git`
  - !GitHub 저장소를 원격 저장소로 등록합니다.
- `git push -u origin main`
  - 로컬의 main 브랜치를 GitHub에 업로드합니다.
